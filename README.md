# Data Cleaning & Visualization Project

## Overview
This project focuses on cleaning raw data and creating meaningful visualizations using Python.

## Features
- Loaded CSV dataset
- Removed duplicate records
- Handled missing values
- Cleaned the dataset
- Created visualizations using Matplotlib and Seaborn

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Output
- Cleaned dataset
- Data visualization charts
